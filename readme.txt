== Description ==

This plugin requires Twitter Tools. It will shorten all tweeted URL's using the Tynie.net service.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `pxs_url_short.php` to the `/wp-content/plugins/` directory
1. Make sure you are using Twitter Tools plugin
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done
